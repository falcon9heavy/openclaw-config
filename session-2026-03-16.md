## OpenClaw — Heartbeat Cost Fix (CRITICAL)

**Problem:** OpenClaw ships with heartbeat enabled by default (every 30 min) using the primary model. Our primary was `claude-opus-4-6` — one of the most expensive models. This was silently making 48+ Opus API calls per day, draining credits and eventually triggering a billing cooldown (9 hours).

**Root cause:** Default heartbeat config + expensive primary model + no awareness of the feature.

**Fix applied:**
```bash
openclaw config set agents.defaults.heartbeat.every "0m"
openclaw gateway restart
```

**Key:** The config key is `every` with value `"0m"` to disable. Other approaches (`enabled: false`, `interval: 0`, `heartbeat: false`) are NOT valid in this version.

**Additional fix:** Removed useless Bedrock fallback models (no AWS credentials configured) that were spamming error logs.
