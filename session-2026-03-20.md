## OpenClaw — Heartbeat Cost Fix

**Problem:** Heartbeat enabled by default (every 30 min) using Opus. Silently making 48+ expensive API calls/day, drained credits, triggered billing cooldown.

**Fix:**
```bash
openclaw config set agents.defaults.heartbeat.every "0m"
openclaw gateway restart
```

**Key learning:** The only valid way to disable heartbeat is `every: "0m"`. Other approaches (`enabled: false`, `interval: 0`, `heartbeat: false`) are NOT valid. Took multiple failed attempts before checking the docs — lesson learned.

## OpenClaw — Dashboard & Architecture Orientation

Explored the OpenClaw "OS" analogy in depth:
- **Gateway** = kernel (always running, manages sessions, routes messages)
- **Channels** (Telegram, WebChat) = I/O devices
- **Skills** = installed software
- **Cron jobs** = scheduled tasks
- **Sub-agents** = child processes
- **Workspace** = agent's home directory with file-based memory
